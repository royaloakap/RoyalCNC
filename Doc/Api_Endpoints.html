Admin API Endpoints

Add a new user to the database

URL: http://localhost/admin/add_user?username=&password=&username_to_add=&password_to_add=&preset_to_use=

preset_to_use options:

String: All presets in the assets/config/presets.json "Plans" section.

Response:

{
    "error": false/true,
    "message": "API response will be here."
}

Edit a user in the database

URL: http://localhost/admin/edit_user?username=&password=&user_to_edit=&field_to_edit=&new_value=

user_to_edit options:

String: All users that exist in the database.

field_to_edit options:

Integer: Cooldown, MaxTime, Concurrents, DaysActive
Bool: VIP, Reseller, Admin, Api, Powersaving, Bypassblacklist, banned

Response:

{
    "error": false/true,
    "message": "API response will be here."
}

Delete a user from the database

URL: http://localhost/admin/delete_user?username=&password=&user_to_delete=

user_to_delete options:

String: All users that exist in the database.

Response:

{
    "error": false/true,
    "message": "API response will be here."
}

View a specific user's attack logs in the database

URL: http://localhost/admin/view_user_logs?username=&password=&user_to_view=

user_to_view options:

String: All users that exist in the database.

Response:

{
    "error": true,
    "message": "API response will be here."
}
OR
{
    "error": false,
    "user_logs": [{},{},{}]
}
